probe-rs

a debugging toolset and library for debugging ARM cores on a separate host

why?

  • suboptimal tooling
  • tooling is Rust-unaware
  • too much time

what?

  • rust library for CoreSight
  • implement the ADI 5.2 specification
  • cargo-flash as a rust idiomatic tool to flash MCUs
  • cli tool to inspect microchip memory and debug them
  • vscode plugin to debug microchips

current State

  • read and write memory from ARM cores
  • support for ST-Link v2(v3 soon?) & CMSIS-DAP
  • can halt, step and bp the connected core M0 or M4
  • can flash a binary on to a specific set of MCUS
  • vscode plugin to show stacktrace and variables

outlook

  • parse and run CMSIS-Pack flash algorithm blobs
  • vscode plugin support for breakpoints and SVD based register inspection
  • libjlink support
  • code cleanup, docs, maybe async?
